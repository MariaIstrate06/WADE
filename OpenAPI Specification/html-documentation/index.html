Interface

Up 
get /trafficSigns
gets traffic signs (getTrafficSigns)
Get all traffic signs recorded in the database

Return type

array[TrafficSign]

Example data

Content-Type: application/json
[ {
  "name" : "Speed Limit 50",
  "coordinates" : {
    "latitude" : 69.142536,
    "longitude" : -79.982222
  },
  "description" : "On the road portion following this sign you should have a speed of 50km/h or lower.",
  "id" : "d290f1ee-6c54-4b01-90e6-d701748f0851"
}, {
  "name" : "Speed Limit 50",
  "coordinates" : {
    "latitude" : 69.142536,
    "longitude" : -79.982222
  },
  "description" : "On the road portion following this sign you should have a speed of 50km/h or lower.",
  "id" : "d290f1ee-6c54-4b01-90e6-d701748f0851"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json
  • text/html

Responses

200

traffic signs recorded in the database

204

no recorded traffic signs

default

error payload ErrorModel

Internal

Up 
post /trafficSigns
adds a TrafficSign item (addTrafficSign)
Adds an item to the system

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

body TrafficSign (optional)
Body Parameter — TrafficSign item to add

Return type

TrafficSign

Example data

Content-Type: application/json
{
  "name" : "Speed Limit 50",
  "coordinates" : {
    "latitude" : 69.142536,
    "longitude" : -79.982222
  },
  "description" : "On the road portion following this sign you should have a speed of 50km/h or lower.",
  "id" : "d290f1ee-6c54-4b01-90e6-d701748f0851"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • applocation/json
  • text/html

Responses

201

item created TrafficSign

400

invalid input, object invalid

409

an identical item already exists

default

error payload ErrorModel
